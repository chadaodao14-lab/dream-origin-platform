## 设计理念

### 核心原则
- **效率优先**：快速定位信息，一键完成操作
- **数据驱动**：清晰的数据展示和可视化分析
- **安全可控**：明确的操作反馈和风险提示
- **一致性**：统一的设计语言和交互模式

### 用户角色
- **平台管理员**：全权管理平台所有功能
- **财务审核员**：审核入金和提现申请
- **项目经理**：管理创业项目和里程碑
- **数据分析员**：查看统计报表和分析数据

## 管理员主面板

### 页面概述
管理员后台的首页，展示平台关键指标和快速操作入口。

### 布局结构
```
┌─────────────────────────────────────────────┐
│ 管理员后台          [返回用户中心]           │
├─────────────────────────────────────────────┤
│ 关键指标卡片区                               │
│ ┌──────────┐ ┌──────────┐ ┌──────────┐     │
│ │ 总用户数 │ │ 总入金数 │ │ 入金总额 │     │
│ │    2     │ │    0     │ │ $0.00    │     │
│ └──────────┘ └──────────┘ └──────────┘     │
│ ┌──────────┐                                 │
│ │ 待审核数 │                                 │
│ │    0     │                                 │
│ └──────────┘                                 │
├─────────────────────────────────────────────┤
│ 管理功能卡片区（3列网格）                    │
│ ┌──────────────┐ ┌──────────────┐ ┌───────┐│
│ │ 💰 入金管理  │ │ 👥 用户管理  │ │ 📊   ││
│ │ 描述文本     │ │ 描述文本     │ │ 描述  ││
│ │ [管理入金]   │ │ [管理用户]   │ │ [管理]││
│ └──────────────┘ └──────────────┘ └───────┘│
│ ... 更多卡片 ...                             │
├─────────────────────────────────────────────┤
│ 最近活动区                                   │
│ ┌─────────────────────────────────────────┐ │
│ │ 最近处理的入金申请、项目审批等           │ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

### 关键指标卡片
- **总用户数**：平台注册用户总数
- **总入金数**：用户提交的入金申请总数
- **入金总额**：所有入金申请的总金额
- **待审核**：等待审核的入金申请数量

### 管理功能卡片（9个）
1. **入金管理** - 审核用户入金申请，确认后自动触发分润计算
2. **用户管理** - 查看用户信息，管理激活状态和角色权限
3. **项目管理** - 审批创业项目，管理项目状态和里程碑
4. **提现管理** - 处理用户提现申请，生成交易哈希
5. **慈善基金** - 管理慈善基金余额，创建捐赠记录
6. **数据统计** - 查看平台数据统计和分析报表
7. **资金池监控** - 监控资金分布、流动和风险
8. **系统设置** - 配置平台参数和基本信息
9. **权限管理** - 管理管理员账户和权限矩阵

### 交互设计
- 点击卡片进入对应功能页面
- 关键指标实时更新
- 最近活动显示最新的5条操作记录

├─────────────────────────────────────────────┤

│ └──────────────────────────────────────────┘│

### 报表类型

#### 1. 用户统计报表
- **用户增长趋势**：折线图展示用户增长趋势
- **用户统计表**：注册用户、激活用户、流失用户、活跃度

#### 2. 收入统计报表
- **收入来源分布**：饼图展示入金、分润、项目收益、其他收入的占比
- **收入趋势**：折线图展示收入变化趋势
- **收入统计表**：各类收入的金额和占比

#### 3. 分润统计报表
- **分润分布**：柱状图展示各层级分润金额
- **分润趋势**：折线图展示分润变化趋势
- **分润统计表**：各层级分润金额、人数、平均分润

#### 4. 项目统计报表
- **项目状态分布**：饼图展示进行中、已完成、已失败项目的占比
- **项目进度**：柱状图展示各项目的投资进度
- **项目统计表**：项目数量、总金额、平均进度

### 时间范围选择
- **日**：显示今天的数据
- **周**：显示本周的数据
- **月**：显示本月的数据
- **自定义**：选择自定义时间范围

## 通用设计规范

### 按钮设计
| 类型 | 颜色 | 用途 |
|------|------|------|
| 主按钮 | 蓝色 | 主要操作（确认、保存、提交） |
| 成功按钮 | 绿色 | 成功操作（批准、激活） |
| 危险按钮 | 红色 | 危险操作（拒绝、删除、禁用） |
| 次按钮 | 灰色 | 次要操作（返回、取消） |

### 表格设计
- **行高**：48px
- **字体**：14px
- **边框**：浅灰色（#E5E7EB）
- **斑马纹**：奇数行背景色为 #FFFFFF，偶数行为 #F9FAFB
- **悬停**：背景色变为 #F3F4F6

### 表单设计
- **输入框高度**：40px
- **字体**：14px
- **边框**：1px 浅灰色
- **焦点**：蓝色边框 + 蓝色阴影
- **错误**：红色边框 + 红色提示文本

### 模态框设计
- **背景**：半透明黑色（rgba(0,0,0,0.5)）
- **宽度**：500px（最大）
- **圆角**：8px
- **阴影**：0 20px 25px -5px rgba(0,0,0,0.1)

### 加载状态
- **加载动画**：旋转的加载图标
- **加载文本**："加载中..."
- **禁用状态**：按钮变灰，不可点击

### 空状态
- **图标**：空状态图标
- **文本**："暂无数据"或"暂无记录"
- **操作**：可选的"创建"或"返回"按钮
